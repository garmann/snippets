## iterm
word jumping with cursor
- goto: settings > profiles -> keys
- add/plus
- left jump:
  - Shortcut: alt + left
  - Action: Send Escape Sequence
  - Esc+: b
- right jump:
  - Shortcut: alt + right
  - Action: Send Escape Sequence
  - Esc+: f
